# Lab6 - Huong dan chay app

README nay huong dan chay giao dien app truoc.
Hien tai **chua can chay phan crawl o backend**.

## Cau truc chinh

- `frontend/`: giao dien web (HTML, CSS, JS)
- `backend/crawl/crawl.py`: script crawl lich kham (tam thoi bo qua)

## Yeu cau

- Python 3.x (de tao web server local nhanh)
- Trinh duyet (Chrome/Edge/Firefox)

## Cach chay app (frontend)

1. Mo terminal tai thu muc goc du an (`src`).
2. Chay lenh:

```powershell
cd frontend
python -m http.server 3000
```

3. Mo trinh duyet va truy cap:

```text
http://localhost:3000
```
